# PlaygroundX

**PlaygroundX** is a modern full-stack web application built with **TypeScript**, **Vite**, **React**, and **Supabase**. It provides interactive social features, real-time rooms, games, and creator tools.  

---

## Features

- User authentication and profiles
- Real-time chat and activity rooms
- Games and interactive experiences
- Creator dashboard and content management
- Payments and wallet management
- Admin panel for managing users, reports, and settings
- Fully responsive and modern UI built with Tailwind CSS

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS  
- **Backend / Database:** Supabase (PostgreSQL, real-time subscriptions)  
- **State Management:** React hooks, Context API  
- **Version Control:** Git & GitHub
